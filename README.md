# NodeJs Project

we have to install nodejs and nodemon to run our project

to install nodemon we can use by typing the fallowing command in our terminal:

npm install nodemon -g

this command will install nodemon global

for more info please look at the fallowing tutorial:

https://www.digitalocean.com/community/tutorials/workflow-nodemon
